# AI Dashboard Builder

Build beautifuldata dashboard with AI. Upload your data, ask questions in natural language, and get instant visualisations.

## Tech Stack
- Frontend: Next.js 14, React, TypeScript, Zustand, Tailwind CSS, Prisma
- Backend: Python, FastAPI, LlamaIndex
- Database: PostgreSQL, PGVector
- AI: Google Gemini
- DevOps: Docker Compose, GitHub Actions, Vercel, Render
